# 🚀 Quick Start Guide

Get your portfolio live in **5 minutes!**

## ⚡ 60-Second Setup

### Step 1: Download Files (30 seconds)
Download these 3 files:
- `aviral-portfolio.html`
- `aviral-styles.css`
- `aviral-script.js`

Save them in the **same folder**.

### Step 2: Test Locally (15 seconds)
1. Double-click `aviral-portfolio.html`
2. Your portfolio opens in browser
3. Test navigation and form
4. ✅ Works perfectly!

### Step 3: Deploy to GitHub (5 minutes)

1. **Go to GitHub**
   - Visit https://github.com
   - Sign in (or create free account)

2. **Create Repository**
   - Click `+` → "New repository"
   - Name: `aviralcodes29.github.io`
   - Make it "Public"
   - Click "Create repository"

3. **Upload Files**
   - Click "uploading an existing file"
   - Drag & drop your 3 files
   - Click "Commit changes"

4. **Rename HTML File**
   - Click `aviral-portfolio.html`
   - Click pencil icon to edit
   - Change name to `index.html`
   - Click "Commit changes"

5. **Visit Your Portfolio**
   - Wait 1-2 minutes
   - Go to: `https://aviralcodes29.github.io`
   - 🎉 Your portfolio is LIVE!

## 🎯 Common Customizations

### Change Your Name
Open `aviral-portfolio.html` in text editor:
- Find: `Aviral Singh`
- Replace with: `Your Name`
- Save and re-deploy

### Change Colors
Open `aviral-styles.css`, find top section:
```css
:root {
    --accent-cyan: #00d4ff;     /* Change this color */
    --accent-purple: #a855f7;   /* Or this one */
}
```

### Add New Project
In `aviral-portfolio.html`, find Projects section:
```html
<div class="project-card">
    <div class="project-header">
        <h3>Your Project Name</h3>
        <span class="project-icon">🎯</span>
    </div>
    <p class="project-desc">Your description</p>
    <div class="project-tech">
        <span class="tech-tag">Technology</span>
    </div>
    <a href="https://your-link.com" target="_blank" class="project-link">
        Live Demo <span>→</span>
    </a>
</div>
```
